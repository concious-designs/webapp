Instructions

Raven's Progressive Matrices (RPM) tests are used to gauge intelligence. This test is comprised of multiple problem sets, each of which contain 12 problems. Each problem itself contains either a 2x2 or 3x3 array of related figures. The figure in the bottom-right corner is any one of six possible answer choices. One of the available answer choices - shown to the right of the problem figures - will complete the set of problem figures. An example is shown below. Hovering over the question mark figure will display the correct answer.
